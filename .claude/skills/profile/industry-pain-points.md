# 行业痛点库

Profile skill 在匹配行业痛点时引用此文件。根据企业所属行业选择对应痛点库。

---

## 科技/软件企业

| 痛点领域 | 典型表现 | 数字化机会 |
|----------|----------|------------|
| **项目管理** | 多项目并行混乱、进度不透明、资源冲突 | 项目管理平台、资源排程、看板工具 |
| **客户管理** | 客户信息分散、销售过程不透明、续约率低 | CRM、客户成功平台 |
| **知识管理** | 经验在个人脑中、新人培养慢、重复造轮子 | 知识库、文档协作、AI助手 |
| **交付管理** | 需求变更频繁、交付延期、验收扯皮 | 需求管理、交付看板、电子签章 |
| **回款管理** | 账期长、回款难跟踪、坏账风险 | 合同管理、应收管理、收款自动化 |
| **产品管理** | 需求散落、迭代不规划、版本混乱 | 产品管理系统、需求管理工具 |
| **运维管理** | 故障响应慢、监控不完善、排障困难 | 监控告警系统、运维工单、故障管理 |

## 制造业

| 痛点领域 | 典型表现 | 数字化机会 |
|----------|----------|------------|
| **生产管理** | 排产靠Excel、进度不透明、插单混乱 | MES、APS排程、生产看板 |
| **质量管理** | 质检靠人工、追溯困难、客诉处理慢 | QMS、条码追溯、SPC |
| **库存管理** | 账实不符、呆滞料多、缺料停产 | WMS、智能补货、库存预警 |
| **成本核算** | 成本算不清、报价靠经验、利润黑箱 | 成本核算系统、报价工具 |
| **设备管理** | 故障突发、维护靠经验、OEE低 | 设备点检、预测性维护、OEE分析 |
| **订单管理** | 订单靠电话/微信、交期难跟踪 | OMS、客户门户 |
| **供应链管理** | 供应商分散、对账困难、质量追溯 | SRM、供应商门户、供应链协同 |
